
# Personal Expense Tracker (Java)
A simple console-based Java application to track and manage daily expenses.
## 📌 Features
- Add daily expenses (amount, category, date)
- Display all expenses
- Monthly expense report
- Highest spending category
- Save & load data using file handling
##  Technologies Used
- Java
- OOP Concepts
- ArrayList & HashMap (Collections)
- File Handling
- LocalDate API
##  Purpose
This project demonstrates basic Java programming, OOP principles, and real-world expense tracking functionality.
